# STUDENT-RECORD-DATABASE-PROJECT

ABSTRACT:
The Student Record Database System is a simple console-based project developed using the C programming language to manage student records efficiently. The system allows users to add, update, search, delete, and display student details through a menu-driven interface. Linked lists are used to store student records dynamically, and file handling is used to save and maintain records permanently.

This project helps reduce manual work and improves the organization of student information in a simple and efficient way. It also demonstrates the practical implementation of structures, pointers, linked lists, functions, and file handling concepts in C programming.

OBJECTIVES:
* To maintain student records efficiently.
* To reduce manual work in student data management.
* To implement linked list operations in C.
* To perform add, update, search, delete, and display operations.
* To store student records permanently using file handling.
* To improve programming skills using real-time applications.

FUNCTIONS USED IN THE PROJECT:
1.add_student():
This function is used to add new student records into the system. It stores details such as student ID, name, marks, department, and phone number in the linked list.

2.update_student():
This function is used to modify existing student details. Users can update student information whenever required.

3.delete_student():
This function removes student records from the database using the student ID or name.

4.search_student():
This function is used to search and display student details based on student ID or name.

5.display_students():
This function displays all student records stored in the system.

6.save_data():
This function saves all student records permanently into files using file handling.

7.read_data():
This function reads student records from files and loads them into the program during execution.

CONCLUSION:
The Student Record Database System is useful for managing student information in a simple and organized way. The project demonstrates the practical use of linked lists, structures, pointers, functions, and file handling in C programming. It helps improve programming knowledge and provides experience in developing real-time applications.
